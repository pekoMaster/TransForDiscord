為什麼選擇 Peko Embed？

Discord 內建預覽經常壞掉、缺圖、不顯示影片。Peko Embed 幫你把每條連結都變成完美的預覽卡片。

🔗

網址預覽修正

自動偵測訊息中的 URL，產生增強的預覽 Embed。圖片、影片、文字一個不漏。告別 Discord 內建的破碎預覽。

🌐

一鍵推文翻譯

看到日文推文不用再切分頁！按一下翻譯按鈕，AI 幫你翻成中文。支援 DeepL 及多種 AI 翻譯引擎。

🖼️

多圖分頁瀏覽

Pixiv 多圖作品、PTT 多圖文章，不用點進去就能在 Discord 裡翻頁瀏覽。支援動圖(Ugoira)轉 MP4。

🎬

影片直接播放

Twitter 影片、Bilibili 影片直接在 Discord 內嵌播放，不用另開瀏覽器。支援 V2 Components 原生媒體展示。

🛡️

防爆雷系統

一鍵將預覽內容隱藏為防爆雷模式，保護還沒看到最新回的同好。適合追番、追劇、追 Vtuber 企劃的伺服器。

⚡

輕量無負擔

純 URL 預覽功能，不需要複雜設定。加入伺服器後自動運作，不干擾其他機器人，不儲存你的訊息內容。

🌐 推文一鍵翻譯為中文

Vtuber 粉絲最需要的功能 — 推主發了新推、切了新片，按一下就能看懂內容。

為中文使用者而生

追 Hololive、Nijisanji、個人勢？再也不用等有愛翻譯。Peko Embed 讓你的伺服器即時看懂日文推文。

  • 支援 DeepL 高品質翻譯
  • 支援 Gemini / GPT / Claude 等 AI 翻譯
  • 可綁定自己的 API Key，翻譯不限量
  • 翻譯結果跨頻道共享快取，省 API 用量
  • 翻譯按鈕內建於預覽卡片，一鍵切換原文/譯文
原文 (日本語)
今日もお疲れぺこ～！配信来てくれたみんなありがとう！明日はコラボだから楽しみにしててぺこよ🐰

今天也辛苦了peko～！感謝來看直播的大家！明天有聯動所以請期待一下peko喔🐰
