服務條款

最後更新：2026-05-10

歡迎使用 Peko Embed（前稱 TFD / TransForDiscord）。在邀請或使用本機器人前，請仔細閱讀以下條款。邀請本機器人到您的伺服器，或在已加入的伺服器中使用本機器人，即視為您同意本條款全文。

一、服務說明

Peko Embed 是一個 Discord 機器人，提供以下核心功能：

  • 自動偵測訊息中支援平台的 URL（Twitter/X、Pixiv、PTT、Threads 等）並產生增強的預覽
  • 透過按鈕提供推文/貼文翻譯（DeepL / 使用者自備 AI Key）
  • 多圖預覽分頁、防爆雷、媒體合併等輔助功能

二、使用條件

2.1 年齡限制

您須年滿 13 歲（或您所在地區法定 Discord 使用最低年齡）才能使用本服務。

2.2 必要權限

本機器人運作需要以下 Discord 權限：

  • View Channels / Send Messages / Embed Links
  • Manage Webhooks（用於以使用者身份模擬發送預覽）
  • Read Message History
  • Message Content Intent（特權 Intent）

伺服器管理員可隨時撤銷上述權限或將機器人踢出伺服器。

2.3 禁止行為

使用本服務時，您不得：

  • 透過刷量方式濫用本服務（包含但不限於連續貼相同 URL 觸發預覽、利用機器人進行爬蟲）
  • 將本服務用於騷擾、誹謗、傳播違法內容、規避平台限制等不當目的
  • 嘗試逆向工程、破解加密、攻擊本服務基礎設施
  • 將本服務用於商業目的而未獲得明確授權

違反本條款的行為將導致您或您的伺服器被加入排除清單，嚴重情況下機器人將拒絕在您的伺服器繼續運作。

三、自備 API Key（AI 翻譯）

3.1 範圍

您可透過 /pe api add 提供您的 OpenAI / Anthropic / Google Gemini API Key 以使用 AI 翻譯功能。

3.2 您的責任

  • 您對自備 Key 的所有 API 用量自行負責，包含費用、配額管理
  • 您確保有權使用該 Key，未違反對應服務商的條款
  • 您可隨時 /pe api del 移除 Key

3.3 我們的責任

  • 我們以 AES-256-GCM 加密儲存您的 Key，僅在您觸發翻譯時解密使用
  • 我們不會將您的 Key 用於翻譯以外的用途
  • 我們不會將您的 Key 傳送給第三方（除翻譯時呼叫對應 AI 服務商）

四、內容免責聲明

本服務所產生之預覽內容（推文、貼文、圖片）皆來自第三方平台。本服務不擁有、不審查、不為這些內容的合法性或正確性負責。如有侵權或內容問題，請洽原內容平台。

4.2 翻譯品質

AI/機器翻譯結果可能不精確，僅供參考，本服務對翻譯品質不作保證。

4.3 R18 / NSFW 內容

  • 本服務公開版預設不處理 Pixiv R18 內容
  • 本服務不主動偵測或過濾其他平台的 NSFW 內容；伺服器管理員需自行設定 NSFW 頻道
